# TubeCut Pro — оптимизация раскроя профтрубы

Веб-приложение для технолога/цеха:

- выбор сечения профиля (ширина × высота), длины хлыста и толщины диска;
- детали с количеством и углами реза (45°, 90° и любые в диапазоне);
- расчет эффективной длины с учетом косых резов и пропила;
- упаковка деталей по хлыстам (FFD) для минимизации отхода;
- realtime визуализация деталей и раскроя с фиксированной визуальной длиной элементов;
- обновленный UI в стиле dashboard, подготовленный для точного стилизования под Figma.

## Запуск

```bash
python3 -m http.server 4173
```

Откройте `http://localhost:4173`.
